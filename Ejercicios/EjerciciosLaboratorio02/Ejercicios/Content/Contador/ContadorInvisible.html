Los pseudo-elementos :before y :after que usamos constantemente en CSS no funcionan si se aplican directamente a un elemento de imagen (tipo )

El motivo es que , al igual algunos otros elementos "especiales" son lo que la especificación HTML denomina "elementos reemplazados". Un elemento de este tipo tiene sus dimensiones y apariencia determinados a partir de un recurso externo, en este caso una imagen

Esto, de todos modos, tiene muy fácil solución, pues basta con envolver a la imagen o elemento reemplazado en otro elemento no-reemplazado (generalmente un

) y aplicarle a éste el pseudo-elemento que nos interese.
